# 学习建议
过一遍学习重点，然后在做页面的过程，来巩固CSS知识。常翻 [CSS参考手册](http://css.doyoe.com/)。

## 学习重点
* 选择器。重点学：id选择器 ，类名选择器，层次选择器，子元素选择器，伪类选择器，伪元素选择器，:nth选择器。
* 布局
  * 一些概念：元素类型，盒模型等
  * Flex 布局
  * inline-block 布局
  * positon:static,relative,absolute,fixed
  * 响应式布局：媒体查询和rem
* 样式规则。重点是学下面几类的，其他了解即可
  * 背景(background)
  * 文本(text)
  * 字体(font)
  * 变换(transform)
  * 过渡(transition)
  * 动画(animation)
* 了解样式的继承和层叠
  * 哪些属性是会继承的
  * 样式的优先级
